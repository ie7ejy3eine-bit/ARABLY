مرحباً بك في نظام ARABLY ERP
============================

شكراً لاختيارك نظامنا لإدارة أعمالك.

ملاحظات هامة:
--------------
- عند تشغيل البرنامج لأول مرة، سيطلب منك اختيار أو إضافة شركة.
- اسم المستخدم وكلمة المرور الافتراضيين هما:
  - اسم المستخدم: admin
  - كلمة المرور: admin
- يوصى بشدة بتغيير كلمة المرور الافتراضية بعد أول تسجيل دخول من خلال شاشة "إدارة المستخدمين".

للحصول على الدعم الفني أو الإبلاغ عن مشكلة، يرجى زيارة موقعنا الإلكتروني.
